Port of Filters concept to Nette\Database.


## Install

Via Composer:

```sh
$ composer require zenify/nette-database-filters
```

Register extensions you need in `config.neon`:

```yaml
extensions:
	- Zenify\NetteDatabaseFilters\DI\NetteDatabaseFiltersExtension
```
